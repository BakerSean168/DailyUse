### 快速入门指南

```markdown
# DailyUse 快速入门

欢迎使用 DailyUse！这份指南将帮助你在 5 分钟内开始使用应用。

## 安装

### macOS

1. 从官网下载 `.dmg` 文件
2. 打开下载的文件
3. 将 DailyUse 拖入应用程序文件夹
4. 首次打开时，右键点击应用选择"打开"

### Windows

1. 从官网下载 `.exe` 安装程序
2. 运行安装程序
3. 按照向导完成安装
4. 从开始菜单启动 DailyUse

### Linux

```bash
# Ubuntu/Debian
sudo dpkg -i dailyuse_*.deb

# Fedora/RHEL
sudo rpm -i dailyuse_*.rpm

# 或使用 AppImage
chmod +x DailyUse-*.AppImage
./DailyUse-*.AppImage
```

## 首次启动

首次启动应用时，你会看到欢迎向导：

1. **设置语言** - 选择你喜欢的界面语言
2. **选择主题** - 选择亮色或暗色主题
3. **导入数据** - 如果你有之前的数据，可以在此导入
4. **完成设置** - 点击"开始使用"进入主界面

## 界面概览

![界面概览](./images/interface-overview.png)

1. **侧边栏** - 快速访问各个功能模块
2. **主内容区** - 显示当前模块的内容
3. **顶部栏** - 搜索和快捷操作
4. **状态栏** - 显示专注状态和通知

## 创建你的第一个任务

1. 点击侧边栏的"任务"图标
2. 点击右上角的"+"按钮
3. 输入任务标题，如"完成项目报告"
4. （可选）添加描述、截止日期、优先级
5. 点击"保存"

恭喜！你已经创建了第一个任务。

### 任务管理指南

```markdown
# 任务管理指南

任务是 DailyUse 的核心功能。本指南将详细介绍如何有效管理你的任务。

## 创建任务

### 基本创建

点击"+"按钮或使用快捷键 `Cmd/Ctrl + N` 打开新建任务对话框。

### 必填信息

- **标题** - 简洁描述任务内容

### 可选信息

- **描述** - 任务的详细说明
- **优先级** - 低、中、高
- **截止日期** - 任务需要完成的日期
- **标签** - 用于分类和筛选
- **关联目标** - 将任务关联到某个目标

### 快速添加

使用快速添加功能可以更快地创建任务：

```
快速添加语法：
任务标题 #标签 !高 @明天

示例：
完成报告 #工作 !高 @周五
```

## 任务状态

任务有四种状态：

| 状态 | 说明 |
|------|------|
| 待处理 | 新创建或未开始的任务 |
| 进行中 | 正在处理的任务 |
| 已完成 | 完成的任务 |
| 已取消 | 不再需要完成的任务 |

### 更改状态

- 点击任务前的复选框快速完成
- 在任务详情中选择其他状态
- 使用快捷键切换状态

## 任务视图

### 列表视图

默认视图，按时间或优先级排列任务。

### 看板视图

将任务按状态分栏显示，支持拖拽移动。

### 日历视图

在日历上查看带截止日期的任务。

## 筛选和排序

### 筛选选项

- 状态：待处理、进行中、已完成
- 优先级：高、中、低
- 日期：今天、本周、已过期
- 标签：按标签筛选
- 目标：按关联目标筛选

### 排序选项

- 创建时间
- 截止日期
- 优先级
- 标题

## 任务操作

### 编辑任务

点击任务打开详情面板，进行编辑。

### 复制任务

右键点击任务，选择"复制任务"。

### 删除任务

右键点击任务，选择"删除任务"，确认后删除。

### 批量操作

1. 按住 Shift 选择多个任务
2. 使用批量操作菜单
3. 可批量更改状态、优先级或删除

## 子任务

将复杂任务分解为子任务：

1. 打开任务详情
2. 点击"添加子任务"
3. 输入子任务标题
4. 完成子任务会自动更新主任务进度

## 任务提醒

### 设置提醒

1. 在任务详情中点击"提醒"
2. 选择提醒时间
3. 设置提醒方式（通知/邮件）

### 提醒选项

- 截止前提醒：1小时、1天、1周
- 自定义时间
- 重复提醒

## 最佳实践

### 每日规划

1. 每天早上查看今日任务
2. 调整优先级
3. 预估所需时间
4. 安排专注时间

### 每周回顾

1. 查看本周完成情况
2. 清理过期任务
3. 规划下周任务
4. 调整目标进度

### GTD 方法

1. **收集** - 快速记录所有想法
2. **整理** - 添加标签和优先级
3. **执行** - 按计划完成任务
4. **回顾** - 定期检查和调整
```

### 目标管理指南

```markdown
# 目标管理指南

目标帮助你专注于重要的事情，并追踪长期进展。

## 创建目标

### 目标类型

- **项目目标** - 有明确结束时间的目标
- **习惯目标** - 需要持续执行的目标
- **里程碑** - 大目标的阶段性成果

### 设置目标

1. 点击侧边栏"目标"
2. 点击"新建目标"
3. 填写目标信息

### 目标属性

- **标题** - 清晰的目标描述
- **描述** - 详细说明
- **类型** - 项目/习惯/里程碑
- **截止日期** - 目标完成时间
- **关键结果** - 可衡量的成果指标

## SMART 目标

设置目标时遵循 SMART 原则：

| 原则 | 说明 | 示例 |
|------|------|------|
| **S**pecific | 具体明确 | "学习 React" → "完成 React 官方教程" |
| **M**easurable | 可衡量 | 添加具体的完成标准 |
| **A**chievable | 可实现 | 在能力范围内 |
| **R**elevant | 相关性 | 与长期目标一致 |
| **T**ime-bound | 有时限 | 设置明确截止日期 |

## 目标进度

### 自动追踪

- 关联任务完成时自动更新
- 子目标进度汇总到父目标

### 手动更新

- 在目标详情页调整进度
- 添加进度备注

### 进度可视化

- 进度条显示完成百分比
- 图表显示历史趋势

## 目标与任务

### 关联任务

将任务关联到目标：
- 创建任务时选择关联目标
- 在目标详情中添加新任务

### 任务贡献

完成关联任务会自动增加目标进度。

## 目标分解

### 创建子目标

将大目标分解为小目标：

1. 打开目标详情
2. 点击"添加子目标"
3. 设置子目标信息

### 层级结构

```
🎯 年度目标
├── 📌 Q1 目标
│   ├── ✅ 子目标 1
│   └── 🔄 子目标 2
├── 📌 Q2 目标
└── 📌 Q3 目标
```

## 目标视图

### 看板视图

按状态查看所有目标。

### 时间线视图

在时间轴上查看目标安排。

### 树形视图

查看目标层级结构。

## 目标回顾

### 定期回顾

设置定期回顾提醒：
- 每周回顾进度
- 每月评估调整
- 季度总结规划

### 回顾内容

1. 进度是否符合预期
2. 是否需要调整计划
3. 遇到的困难和解决方案
4. 下一步行动计划
```

### 专注功能指南

```markdown
# 专注功能指南

专注模式帮助你集中注意力，提高工作效率。

## 开始专注

### 快速开始

1. 点击顶部的专注按钮，或
2. 使用快捷键 `Cmd/Ctrl + Shift + F`

### 选择任务

专注前可以选择要完成的任务：
1. 点击"选择任务"
2. 从列表中选择一个任务
3. 开始专注

## 番茄工作法

DailyUse 内置番茄工作法支持：

### 工作周期

- **专注时间** - 默认 25 分钟
- **短休息** - 默认 5 分钟
- **长休息** - 每 4 个周期后，默认 15 分钟

### 自定义时间

在设置中调整各项时长：
1. 打开设置 → 专注
2. 调整专注/休息时长
3. 设置每日目标

## 专注界面

![专注界面](./images/focus-interface.png)

1. **计时器** - 显示剩余时间
2. **进度环** - 可视化进度
3. **当前任务** - 正在专注的任务
4. **控制按钮** - 暂停/继续/结束

## 专注控制

### 暂停

点击暂停按钮或按 `Space` 暂停计时。

### 继续

再次点击或按 `Space` 继续。

### 提前结束

点击结束按钮，可选择是否保存本次记录。

### 跳过休息

休息时间可选择跳过直接开始下一个专注周期。

## 勿扰模式

专注时自动启用勿扰模式：

- 静音通知
- 隐藏窗口通知
- 可设置例外应用

## 统计数据

### 今日统计

- 专注次数
- 总专注时长
- 完成任务数

### 历史记录

- 每日/周/月专注时长
- 专注时段分布
- 效率趋势图

### 成就系统

完成专注目标获得成就：
- 连续专注天数
- 累计专注时长
- 完成任务数量

## 高级功能

### 专注音乐

内置白噪音和专注音乐：
- 雨声
- 咖啡厅
- 自然音效
- 纯音乐

### 屏幕时间

显示专注期间的屏幕使用情况。

### 网站屏蔽

专注时屏蔽指定网站：
1. 打开设置 → 专注 → 屏蔽列表
2. 添加要屏蔽的网站
3. 专注时这些网站将无法访问

## 与其他功能集成

### 与任务集成

- 专注完成后自动更新任务进度
- 记录专注与任务的关联

### 与目标集成

- 专注时长计入目标进度
- 在目标统计中查看专注贡献

### 与日程集成

- 在日程中查看专注记录
- 安排专注时间块
```
