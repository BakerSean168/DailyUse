# 快速创建提醒模板库

## 概述

内置常见提醒模板（如喝水、站立、番茄钟、休息、日报），用户可一键创建，降低上手门槛。

## 目标与价值

- 新用户首周提醒创建数 +50%
- 提升提醒功能的活跃度和留存

## 主要设计点

- 预置 5-10 个常见模板，支持一键导入
- 支持自定义模板库扩展
- 字段对齐 ReminderTemplate，新增 TemplatePreset 表

## MVP 路径

- 前端提供模板库入口，后端提供模板数据接口
- 支持一键批量创建

## 验收标准

- 用户可在 1 分钟内完成模板导入
- 模板字段与 ReminderTemplate 完全对齐
