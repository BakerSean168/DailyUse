## 🎯 学习目标

完成这个章节后，你应该能够：

- ✅ 在本地成功运行项目
- ✅ 理解项目的目录结构
- ✅ 知道如何查找代码和文档
- ✅ 运行测试和代码检查
- ✅ 提交你的第一个 Pull Request

## 💡 学习建议

1. **动手实践** - 不要只看文档，跟着步骤操作
2. **善用搜索** - 在文档中搜索关键词（Ctrl+F）
3. **查看示例** - [[../examples/README|示例代码]]中有大量实战案例
4. **提问** - 遇到问题？在 GitHub Issues 提问
